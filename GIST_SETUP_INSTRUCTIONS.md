# Create Your SuperClaude Gist - 5-Minute Setup

**Quick setup to make SuperClaude framework permanently available on claude.ai/code**

---

## üéØ Goal

Create a GitHub Gist that you can load in any claude.ai/code session to get full SuperClaude framework capabilities.

---

## üìù Step-by-Step Instructions

### Step 1: Create GitHub Gist (2 minutes)

1. **Go to**: https://gist.github.com/

2. **Click**: "Create new gist" button (top right)

3. **Add 3 files** (copy content from `gist-content/` folder):

   **File 1**: `superclaude-framework.md`
   - Description: "SuperClaude Framework - Quick Loader"
   - Content: Copy from `gist-content/superclaude-framework.md`

   **File 2**: `hub-project-context.md`
   - Description: "Hub Automation Project Context"
   - Content: Copy from `gist-content/hub-project-context.md`

   **File 3**: `finderr-project-context.md`
   - Description: "FINDERR Project Context"
   - Content: Copy from `gist-content/finderr-project-context.md`

4. **Set visibility**: Public (so URLs work from any device)

5. **Click**: "Create public gist"

6. **Copy the Gist URL**: e.g., `gist.github.com/[username]/[gist-id]`

---

### Step 2: Create Short URL (Optional - 2 minutes)

**Option A - bit.ly**:
1. Go to https://bit.ly
2. Paste your Gist URL
3. Create custom link: `bit.ly/superclaude` or `bit.ly/my-framework`
4. Save the short URL

**Option B - TinyURL**:
1. Go to https://tinyurl.com
2. Paste your Gist URL
3. Create custom alias: `tinyurl.com/superclaude-framework`
4. Save the short URL

**Option C - GitHub Short Hash** (automatic):
- Your Gist URL is already shortened by GitHub
- Example: `gist.github.com/username/abc123def`

---

### Step 3: Bookmark for Quick Access (1 minute)

**Desktop Browser**:
1. Open your Gist URL
2. Bookmark to favorites bar
3. Name it: "SuperClaude" or "SC Framework"

**Mobile Browser**:
1. Open your Gist URL in mobile browser
2. Add to home screen or bookmarks
3. Test access from mobile

**Alternative - QR Code**:
1. Go to https://www.qr-code-generator.com/
2. Paste your Gist URL
3. Download QR code image
4. Save to phone for instant scanning

## üÜò Troubleshooting

### "Gist not loading in Claude"
- Check Gist is set to "Public" (not secret)
- Use raw GitHub URL format
- Try copying content directly instead

### "Framework commands not working"
- Verify content was copied completely
- Check for formatting issues

- Go to gist.github.com
- Click your profile ‚Üí "Gists"
