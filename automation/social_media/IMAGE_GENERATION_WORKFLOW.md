- **Fonts**: Modern, readable (minimum 24pt for mobile readability)
- **Logo**: UNTRAPD logo placement consistent

### Option 1: Canva (Recommended for Speed)
**Pros**: Fast, templates available, cloud-based, team collaboration
**Cons**: Less customization than Figma

**Workflow**:
1. Create account: canva.com
2. Select "Instagram Post" template (1080x1080px)
3. Create 5 master templates (one per category)
4. Duplicate and customize for 60 variations
5. Export as JPG (85% quality)
6. Batch download all 60 images

**Time estimate**: 4-6 hours for 60 images
